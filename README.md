# Bus Booking System

A simple console-based Bus Booking System developed using Core Java. This application allows users to view available buses, search buses based on source and destination, and book tickets by selecting the desired bus.

## Features

* View available buses
* Search buses by source and destination
* Book tickets
* Check seat availability
* Update available seats after booking
* Menu-driven console interface

## Technologies Used

* Java (Core Java)
* Object-Oriented Programming (OOP)
* Collections (ArrayList)
* Loops and Conditional Statements
* Scanner Class for User Input

## Project Structure

```text
BusBookingSystem.java
```

## How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd BusBookingSystem
```

3. Compile the program:

```bash
javac BusBookingSystem.java
```

4. Run the program:

```bash
java BusBookingSystem
```
